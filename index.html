Overview

Sequence are aligned end to end along their entire length and are suitable for closely related sequences.

Scoring System Step 1

First of all scoring scheme is decided for whether it is a match or not and if there is a gap. (Default -> Match=2, Not Match=-1, Gap=-1)

DP Initiation Step 2

The first cell of the scoring matrix is set to zero and all columns starting from the second of the first row is set as the sum of the previous cell in that row and gap penalty.

DP Extension Step 3

Starting from the cell A[1][1] and moving to bottom right of the matrix values for each cell is set as MAX(A[i-1][j-1] + (p or q),A[i-1][j] + r, A[i][j-1] + r) depending on match(p) or not a match(q).

TracebackStep 4

The aligned string is obtained by following the traced path and the one with the max score is the optimal alignment.

Overview

Finds local region with the highest level of similarity between the two sequences and is suitable for more divergent sequences.

Scoring System Step 1

Scoring scheme is decided and a scoring matrix is created with dimensions as (sequence1 length + 1) * (sequence2 length + 1).

DP Initiation Step 2

All values in the first row and the first column is set as zero.

DP Extension Step 3

Starting from the cell A[1][1] and moving to bottom right of the matrix values for each cell is set as MAX(A[i-1][j-1] + (p or q),A[i-1][j] + r, A[i][j-1] + r, 0) depending on match(p) or not a match(q).

Traceback Step 4

The aligned string is obtained by following the traced path starting from the cell with the highest value and the one with the max score is the final alignment.
